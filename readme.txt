Tags: mobile, phone
Requires at least: 2.0
Tested up to: 2.6

Tagga Send to mobile plugin allows blog viewers to send post content by SMS to their phone.

== Description ==

Tagga Send to mobile plugin allows blog viewers to send post content by SMS to their phone.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `tagga_send_to_mobile.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How long does it takes messages to send ? =

Within two minutes.
